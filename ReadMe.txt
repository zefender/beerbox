Using Untappd.com this app finds beer, which you bought and want to add to your personal collection, which you have at home.

To add beer into your stash you should tap «search» icon in navigation bar, in the search view you should enter beer or brewery name and wait for search result. When you see the results you can swype left as usually «swype to delete» and add beer to stash. The beer will display in your stash view immediately. You can tap on the beer in stash and get more details about it. 

For personal “beer shelf” you can use 2 options of sorting: by date, by name.

Also, you can find additional information about any beer: description, ratings, reviews, information about brewery.
